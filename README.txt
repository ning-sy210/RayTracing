This project is about raytracing, where objects in the scene are rendered on the screen through real-time raytracing technique to produce a realistic coloring of objects in the scene. 

Goto:
https://www.shadertoy.com/new

Copy and paste the code of "RayTracing.frag" file into the coding interface on the website. 
On the website, click on the small black triangle located on the bottom-left corner of the coding interface to see the scene rendered in real-time raytracing.
